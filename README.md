    BeeYou là một website bán túi tote của nhãn hàng beeyou. Tại đây người dùng có thể thwucj hiện xem, mua các sản phầm túi Tote của nhãn hàng

    Bước 1: cd beeyou

    Bước 2: npm install --save --legacy-peer-deps
    
    Bước 3: npm start
